# WECHIYE – Secure Desktop Expense Tracker

Built with Go + Wails, SQLCipher, and vanilla JavaScript.

## Features
- 🔒 End-to-end encrypted database.
- 👫 Couple sync with QR pairing.
- 📊 Budget tracking with visual insights.
- 🌙 Light/dark mode.
- 🌍 Multilingual (English, Amharic, Oromo).
